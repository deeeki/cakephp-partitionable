h3. Commands

* create
** create partitioning with now date
* remove
** remove partitioning
* add
** add partition arg[0] date
* add_bulk
** add partitions 7 days after a week
* remove
** remove partition arg[0] date
* remove_bulk
** remove partitions 7 days before a week
